# Orbital Strike Dialog System

## Описание
Система управления орбитальными ударами через интерфейс dialog в Minecraft. Позволяет добавлять, удалять и запускать до 10 зарядов с различными типами и уровнями мощности.

## Установка
1. Скопируйте datapack в папку `datapacks` вашего мира
2. Перезагрузите датапак командой `/reload`
3. Система автоматически инициализируется при загрузке

## Использование

### Открытие меню
Используйте команду:
```
/function orbital:open_dialog
```

### Главное меню
В главном меню доступны следующие опции:
- **Add Charge** - Добавить новый заряд
- **Remove Charge** - Удалить заряд из списка
- **Launch Charges** - Запустить все заряды
- **Remove List** - Очистить весь список

### Добавление заряда
1. Нажмите "Add Charge"
2. Введите координаты X, Y, Z или нажмите "Paste Player Coords" для автоматического заполнения
3. Выберите тип заряда:
   - **Stab** - Точечный удар (вертикальная линия TNT)
   - **Nuke** - Ядерный удар (сферический разброс TNT)
4. Выберите уровень мощности:
   - **Power 1** - Базовая мощность
   - **Power 2** - Средняя мощность
   - **Power 3** - Максимальная мощность
5. Нажмите "Add to List" для сохранения

### Удаление зарядов
1. Нажмите "Remove Charge" в главном меню
2. Нажмите кнопку с минусом рядом с зарядом, который хотите удалить
3. Заряд будет удален из списка

### Запуск зарядов
1. Нажмите "Launch Charges" в главном меню
2. Все активные заряды будут запущены последовательно
3. На каждой целевой координате будет выполнен соответствующий орбитальный удар

### Очистка списка
1. Нажмите "Remove List" в главном меню
2. Все заряды будут удалены из списка

## Технические детали

### Типы ударов
- **Stab Power 1-3**: Вертикальная линия TNT (от Y+20 до Y-40)
- **Nuke Power 1-3**: Сферический разброс TNT с различной плотностью

### Ограничения
- Максимум 10 зарядов одновременно
- Координаты должны быть целыми числами
- Чанки автоматически загружаются при запуске

### Хранение данных
Данные хранятся в scoreboards:
- `osc.c0-c9.x/y/z` - Координаты зарядов
- `osc.c0-c9.type` - Тип заряда (0=Stab, 1=Nuke)
- `osc.c0-c9.power` - Уровень мощности (0-2)
- `osc.charge.active` - Флаги активности слотов

## Структура файлов
```
data/orbital/
├── dialog/
│   ├── main.json              # Главное меню
│   ├── add_charge.json        # Форма добавления заряда
│   └── main_remove.json       # Меню удаления
└── function/
    ├── dialog/
    │   ├── init.mcfunction           # Инициализация
    │   ├── open_main.mcfunction      # Открытие главного меню
    │   ├── paste_coords.mcfunction   # Вставка координат игрока
    │   ├── process_add.mcfunction    # Обработка добавления
    │   ├── add_slot_X.mcfunction     # Добавление в слот (0-9)
    │   ├── remove_slot_X.mcfunction  # Удаление из слота (0-9)
    │   ├── launch_charges.mcfunction # Запуск всех зарядов
    │   ├── launch_slot_X.mcfunction  # Запуск из слота (0-9)
    │   └── clear_list.mcfunction     # Очистка списка
    └── open_dialog.mcfunction        # Быстрый доступ к меню
```

## Примеры использования

### Пример 1: Добавление заряда на текущей позиции
1. Откройте меню: `/function orbital:open_dialog`
2. Нажмите "Add Charge"
3. Нажмите "Paste Player Coords"
4. Выберите "Nuke" и "Power 3"
5. Нажмите "Add to List"

### Пример 2: Массовый удар
1. Добавьте несколько зарядов в разных локациях
2. Вернитесь в главное меню
3. Нажмите "Launch Charges"
4. Все заряды будут запущены одновременно

## Совместимость
- Minecraft 1.21+
- Требуется поддержка dialog system (minecraft:notice, minecraft:multi_action)
- Совместимо с существующими функциями orbital strike

## Технические детали dialog системы
- Использует `minecraft:multi_action` для главного меню
- Использует `minecraft:notice` с inputs для форм ввода
- Данные возвращаются через `minecraft:dialog_return` storage
- Поддерживает `minecraft:number_range` и `minecraft:single_option` inputs
